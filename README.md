# Copilot Job Evaluation Agent

**An intelligent, prompt-engineered assistant designed to streamline large-scale job grading and organizational analysis.**

---

## 📋 Project Overview
Conducting job evaluations for 100+ roles is traditionally a time-consuming administrative challenge. This project leverages **Microsoft Copilot** and **Advanced Prompt Engineering** to transform complex interviews into structured data. 

The agent guides you through a standardized framework—evaluating 9 key dimensions such as problem-solving, social skills, and financial responsibility—ensuring consistency and high data integrity.

## Key Features

### Intelligent Context Awareness
The agent actively listens for "contextual shortcuts." If a user mentions a "remote office role" early in the chat, the agent automatically pre-fills the *Physical Environment* factor to Level 1, skipping redundant questions and reducing friction.

### Voice-First Design
Optimized for mobile use, the agent allows users to dictate their job descriptions naturally. It extracts relevant data points from free-form speech, making the process accessible and fast.

### Instant Excel Integration
To eliminate manual data entry, the agent concludes every session by generating:
* A **Markdown Table** for visual confirmation.
* A **Tab-separated raw data string** for seamless copy-pasting directly into an Excel master sheet.

### Standardized Framework
Directly mapped to professional job evaluation levels (Levels 1–7).

## Installation & Usage

1.  **Access the Prompt:** Open `Kravspecar AI assistent.md` in this repository and copy the content.
2.  **Launch Copilot:** Open Microsoft Copilot (M365 or Personal).
3.  **Upload Framework:** Upload the `Faktorbeskrivningar.pdf` (found in this repo) to the chat.
4.  **Initialize:** Paste the instructions and type *"Start Evaluation"*.

* **Efficiency:** Reduces the time per evaluation by an estimated 60%.
* **Scalability:** Specifically engineered to handle 100+ concurrent evaluations without administrative overhead.
